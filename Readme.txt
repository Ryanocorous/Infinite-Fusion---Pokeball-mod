1) Download mod by clicking "Code" then "Download zip"
2) Extract
3) Backup your game folder
4) Open the KIF or PIF file depending on what you've got
5) In another file explorer window, open the unzipped folder including the mods.
6) Inside the extracted mod, open it up and open "KIF" or "PIF" depeding on which you have. 
7) Drag and drop the all files into your game. Click yes on overwriting files.
8) Enjoy.
0) If you want to uninstall, open the uninstall folder (either KIF or PIF) and do the same, drag and drop the files in. This will reset the files.
